# 🚀 FocusDo – Smart Productivity & Focus Tracker

FocusDo is a modern React-based To-Do application designed to improve productivity by combining task management with a built-in Focus Timer. Unlike traditional to-do apps, FocusDo tracks estimated vs actual focus time and provides weekly productivity insights.

---

## ✨ Features

- ✅ Create, Edit and Delete Tasks
- ✅ Mark Tasks as Completed
- 🔍 Search Tasks
- 🎯 Filter Tasks (All, Active, Completed)
- 📂 Category Filter
- 🔥 Priority Levels (High, Medium, Low)
- 🌙 Dark Mode
- ⏱️ Custom Focus Timer
- 📊 Weekly Productivity Chart
- 📈 Accuracy Tracking (Estimated Time vs Actual Time)
- 💾 Local Storage Support
- 📱 Responsive UI

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sampatidotasara/focustodo.git
```

Move into the project

```bash
cd focustodo
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

## 🎯 How It Works

1. Create a task.
2. Assign priority and category.
3. Estimate the time required.
4. Start the Focus Timer.
5. Complete the task.
6. Compare estimated vs actual focus time.
7. View weekly productivity analytics.
